○ ビットマップ形式のファイル *.bmp は，Windowsの表示ソフトによりによ
りディスプレイ上に表示することができる．(* はファイル名)

○ ビットマップ形式のファイルは画像データの他にヘッダがついているので，
処画像理する場合は次のヘッダを取るコマンド hdcut を用いる．また，画像
処理した後，その結果を表示するにはヘッダを付けてビットマップ形式として
表示することができる．ヘッダを付けるコマンドは hdadd である．

・コンパイルの仕方

 hdcut.c のコンパイルは，

	gcc hdcut.c -o hdcut

 のように行う．従って実行ファイル名は hdcut となる．

 hdadd.c のコンパイルも同様に行う．実行ファイル名は hdadd とする．


・使い方

  <*.bmp ファイルのヘッダの取り外し>

	 hdcut *.bmp

　この処理により，*.inc というヘッダの無い binary ファイルができる．こ
のファイルを画像処理する．

 <*.inc ファイルにヘッダを付ける>

	hdadd *.inc

○ ヘッダを取った画像データに対して，画像処理(濃度反転)するプログラム 
inverse.c を付けておくので，binary データの入出力の部分など参考にする．
